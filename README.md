# Passo Verde - Web App 👣🍃
Projeto da disciplina 'Atividade Extensionista' do curso de graduação de Análise e Desenvolvimento de Sistemas

## Sobre o projeto
Aplicação web desenvolvida com foco em educação ambiental, oferecendo informações acessíveis sobre os benefícios da arborização urbana, orientações de plantio e um catálogo de espécies de árvores apropriadas para plantar em calçadas.

O projeto busca aproximar a população de conceitos técnicos relacionados à arborização urbana por meio de uma interface simples, intuitiva e responsiva.

### Funcionalidades
- Breve questionário com perguntas múltipla escolha que realiza recomendações das espécies mais adequadas com base nas características do local informadas pelo usuário; 
- Página com a listagem de todas as espécies, barra de pesquisa por nome e filtros por tipo e tamanho;
- Página individual de cada espécie contendo características gerais e informações básicas voltadas para o plantio em calçadas;
- Sistema de favoritos utilizando localStorage para persistência dos dados no navegador, onde o usuário pode marcar espécies de interesse e acessá-las posteriormente em uma página exclusiva;
- Disponibilização de materiais específicos de cada município;
- Interface projetada com foco em dispositivos móveis, podendo ser instalado pelo próprio navegador e promover uma experiência semelhante a um aplicativo nativo (PWA), através da configuração de um arquivo 'manifest'.
